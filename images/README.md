# 图标说明

请在 images 目录下添加以下图标文件（建议尺寸 81x81 像素）：

## TabBar 图标

### 记录页面图标
- record.png - 未选中状态的记录图标
- record-active.png - 选中状态的记录图标

### 日志页面图标
- logs.png - 未选中状态的日志图标
- logs-active.png - 选中状态的日志图标

## 图标设计建议

可以使用以下工具创建图标：
- 在线图标生成器：https://iconfont.cn/
- 设计软件：Figma、Sketch、Adobe XD
- 图标库：Iconify、Flaticon

## 图标尺寸建议

- TabBar 图标：81x81 像素（推荐）
- 格式：PNG（支持透明背景）
- 颜色：未选中状态使用灰色 (#999999)，选中状态使用蓝色 (#1890ff)

## 临时解决方案

如果没有图标文件，可以暂时从 project.config.json 中移除 tabBar 配置，或者使用纯文字导航。
